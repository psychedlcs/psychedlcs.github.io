新机体验

发表于 2020-06-21 分类于 杂谈

原来使用的Sony VAIO年头已久，近期出现了过热关机现象。加上新专业不少课程需要电脑操作，更换了Lenovo Yoga C740，简单记录一下使用体验。

物理属性

比较有好感的是以下几点：

  • 机身轻盈，符合预计。
  • 屏幕效果显示较好。
  • 电源插口长得很像Type-C（不会就是吧？）

装软件顺序

Yoga预装了Win10正版，遂无需摆弄系统，但是我首先删掉了预装的腾讯视频和酷狗音乐（……）

记录一下其他软件的安装顺序。

1. 搜狗输入法

考虑到后续操作的便捷。

2. Chrome+插件

Chrome到官网下载即可，无需翻墙。

插件是Minyami、Radiko、沙拉划词，去商城搜索直接装即可，还是比较方便的。然后想起来还需要安装IDM。

3. IDM

官网下载+注册即可，之前的正好快到期了，又买了个终身的。

4. QQ

方便传输各种信息。

5. Clash for Windows

安装文件VAIO里有，直接拷贝过来了。订阅URL之前发到过QQ上。

6. Minyami

因为比较麻烦干脆直接装了。

Minyami需要的mkvmerge可以直接从MeGUI中拷贝，省很多时间。

7. 无需安装的视频软件
  • Handbrake
  • 小丸工具箱
  • MeGUI
  • tsMuxeR
8. Typora

写博客用。装完Typora之后安装Hexo。

Hexo搬迁

体感搬迁的过程和第一次安装的过程没有什么区别，移动了以下几个文件和文件夹：

不过现在的上网环境没有之前网速快，导致过程有点曲折。另外差点忘记安装上传到库的插件。

全部搞定后用hexo d -g发布测试博文成功。

无线网络问题

使用大概10分钟左右，就发现有网络时断时续的现象，反映在QQ经常掉线上。排除问题：

1. Clash设置

System Proxy + Rule模式，尝试调成Global、关闭System Proxy、关闭Clash进行试验，没什么改善，应该和代理程序无关。

2. 无线网卡驱动

型号是AX201。去intel官网下了最新驱动，系统内检测也是最新版本，但重启后断网现象仍然存在。

3. 无线网卡属性

网络搜索看到有人说，WIN10会产生这个问题，是因为系统会自动将无线网卡调成省电模式，到设备管理器 - 无线网卡属性里面改一下就好了。

进入属性没找到电源选项设置，但是发现了奇怪的东西：

这里把2.4GHz和5GHz的信号宽度都调为20MHz，之后使用1小时，断网现象未再发生。今天再次开机，开机之间也没有断网，应该已经解决了这个问题。

暂且记录一下，有空的时候再了解背后的原理。

待探索功能

Yoga有平板模式，并且配备了触控笔，还没有试用过。

另外联想预装了一个联想杀毒，一个联想电脑管家，暂时还没有动它们，视情况会考虑卸载。
