客酱的逃跑造诣

发表于 2020-06-24 分类于 麻雀

客酱，23期生，五段，ron2东风凤凰，和勘介并列ron2两天王是我封的。在联盟负责牌谱录入工作，是著名的爱猫人士，经常在推特上晒自己的喵喵。

客酱在ron2现时点的数据是和率23.32%，但铳率只有吓人的9.88%+平均铳点4744，他在逃跑上的造诣由此可见一斑。本篇文章学习客酱的防守型打法。

以防御为轴的打法

客酱的很多选择基于跑路的理念，先看下图的牌：

5对子的时候摸到5s，如果保留最大进张面应该摸切。

不过客酱切了8p兼顾防御。这个巡目2家副露，且见不到dora，即使是断幺一杯，牌形也还凑合，但在客酱心里这是一副需要跑路的牌。

此处客酱拆4p开始跑路，大概是觉得上段s见得不多，下家又在做全带系的牌，9s不太好出。

不过下家已经吃了自己切的9s，还会担心有问题，客酱的小心谨慎可见一斑……当然更合理的解释是9s作为oc，基本上可过3家，客酱在这里为后续留了安全牌。

此处摸成一向听，客酱又切了9s进攻反复横跳。

这里客酱切4p保持好形一向听，并且为了安全不杠7s。

东3局也就是last前，虽然点差不大但自己是个2位，但如果开杠送人dora可能AL要沦落到争4的境地。也就是说在客酱的评判体系里，避4的价值是高于争1的。分奴打法

对家立直，客酱摸到oc出了2枚的1s，毕竟自己听了选择了切3s稳妥弃和。

此处客酱切3p观望情况，不切发，大概是因为场上三元牌太少，客酱认为这张发百分之百会喂到人。相当谨慎了。

另一方面是转染手的打点比较大。

下家切2m后客酱碰了切1m。

此处客酱吃上家的7m，然后切发后付听牌，最终さばき成功。然而我是绝对不会这样听牌的

连他都进攻，你呢

客酱摸东加杠。目前是满贯，杠出来2个就是跳满，直击无条件逆转，在已经听牌的情况下争1的优先级明显高过避4，所以杠了。

连客酱都杠的牌，你又有什么理由不杠呢？

最终没杠出来不过逆2了。

客酱此处吃4m破一发，然后切4p半筋进攻。场上dora出了2枚，自己可见1枚0m，还是庄家，5800点也高于平衡线5200，客酱认为进攻的merit比较大。

0s自摸，连客酱都进攻了，你有什么理由退缩呢？

加速与不加速

这副牌客酱碰5p断幺加速。

这里上家切了dora7p，客酱反而又不碰了。大概是他觉得已经一向听了，万子有2面形，不碰的速度更快。

补充：不碰的话有45678p的进张，也可以拆掉57p，立直dora2就有5200点，和对家同点1位。而碰的话相当于必须要依赖456的进张，残枚已经不多了。

客酱吃急所8p副露，这里以为他想做三色：

但是他摸了6s两面之后又开始北后付，对和率和速度的优先度高于打点。

上家切2p客酱吃听，好形457p高目一气。这个大部分人应该会吃。

3巡拆9s对子做断幺；

第1枚4p没有吃，但是第2枚吃了。鉴于1p是dora，其实他完全可以不吃，可能在客酱心中6巡是一个加速的平衡点。

切什么更安全

继续上图的牌，

下家立直后切了oc的8m，而不是筋牌的3p，其实也是偏守的打法，因为8m过一巡能过2巡。

另外先切9m再切6m，8m的铳率不算特别高，也很难铳对碰。

此处吃上家8p后切4p，精准地扣住了铳牌1s。这是怎么扣的啊

牌感的领域

坎张dora1，场况看起来还行，但是想要更大的打点。客酱先默听1巡观察情况；

对家副露后7p变成了oc，并且速度不容小觑。上家拆了2s对子，不能百分百肯定他在染，但是他很可能进展得差不多了。所以客酱大胆摸切立直！

开挂一发自摸。

客酱吃3m后付4s进攻，铳跳满也掉不到4位，2和3差不多。

相当漂亮的进攻。

总结

客酱的打牌思路虽然相当防守，但值得学习的有以下几个点：

  • 扣铳的精度相当高。
  • 立直成功率相对较高。（体感，因为没有数据可看）因为他不会拿一个没把握的牌去立直，打点不够的时候会拆愚形断幺，在觉得应该加速的时候就会副露。
  • 该进攻时就果断进攻。再怎么防守流，一直跑路的话也是不行的。

客酱的公式规则印象中则平衡很多，不会像红5这样缩。加油保级鸭！
